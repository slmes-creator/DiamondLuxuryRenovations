with dramatic hero imagery, elegant typography, and a dark navy + gold color palette. 
Use TailwindCSS for styling and Montserrat font family with clear hierarchy. 
Apply smooth scroll, hover effects, and gradient overlays for a polished luxury feel.

### Technical SEO Requirements (Non-Negotiable)
- Unique title + meta for every page
- Semantic HTML (header, main, section, footer)
- Image alt text (service + city focused)
- Internal linking between services, blog, contact
- LocalBusiness schema (JSON-LD)
- Review schema on testimonials page
- FAQPage schema on service pages
- XML sitemap + robots.txt
- Fast load (optimized images, lazy loading, minified CSS/JS)
- HTTPS-ready
- Mobile-first responsiveness

### Conversion Optimization
- Sticky header CTA (mobile)
- Click-to-call buttons on mobile
- Form with minimal fields
- Clear “Free Estimate” positioning
- Trust signals near CTAs
- Thank-you page with conversion tracking
